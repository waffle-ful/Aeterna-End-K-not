# 運営費と収支報告

End K not は **GPL-3.0 の無料 Mod** です。全機能は今後も無料で提供され、寄付は一切必須ではありません。「お金より、みなさんのフィードバックが一番の支えです」というのが基本の考え方です。

寄付をいただいた場合、それは**開発者個人の収入ではなく、Mod の運営にかかる実費のみ**に使わせていただきます。使途と収支はこのファイルで公開し、透明性を保ちます。

---

## お金の使い道（ルール）

### 使ってよいもの（実費のみ）

- サーバー・ホスティング費用
- ドメイン費用
- API 利用料（YouTube API クォータ拡張、Mod 専用の AI API 利用 など、用途を特定できるもの）
- イラスト・BGM・効果音などの素材外注
- その他、開発・運営に必要なツールの課金（用途を End K not に特定できるもの）

> **AI 開発支援ツール（Claude など）の定額サブスクについて**: 現在の契約は Mod 開発以外の用途とも兼用のため、基金からは支出せず、**開発者個人の持ち出し（プロジェクトへの現物提供）**として扱います。将来、Mod 専用契約にするか、用途を特定できる形（従量制 API など）に切り替えた場合は、その分を実費として計上できるものとします。

### 使わないもの

- 開発者・運営メンバー個人の生活費、私的な買い物などへの流用
- 広報・渉外の活動費（広告費・接待費 など）— 現時点では各メンバーの持ち出しとし、基金からは支出しません
- 上記「使ってよいもの」に当てはまらない支出

> **広報・渉外の費用について**: 現時点では、広報・渉外にかかる費用は基金から支出せず、各メンバーの持ち出し（プロジェクトへの貢献）として扱います。将来、領収書の出る具体的な広報実費（広告費・宣伝素材 など）を精算する必要が出てきた場合は、事前合意のうえで対象に加えることを検討します。

### 運用ルール

- **保管**: 集まった資金は開発担当が管理し、入出金はすべてこのファイルの台帳に記録します。
- **公開**: すべての収入・支出をこのファイルで公開します。月に一度、締めの残高を報告します。
- **事前合意**: 1 件あたり一定額（現在の基準: **3,000 円**）を超える支出は、事前に運営メンバー間で合意します。
- **私的利用の禁止**: いかなる場合も、資金をメンバー個人の私的目的に使いません。
- **プロジェクト終了・解散時**: 残高はサーバー等の継続運営費に充当します。それも不可能な場合は、メンバーの合意のうえで寄付者への還元または慈善団体への寄付を行います。

> このルールはプロジェクトの成長に合わせて更新することがあります。変更した場合はこのファイルに反映します。

### 経費の立て替え・精算の手順

メンバーが Mod のために立て替えた費用は、口座管理者が精算します。ただし「申告された額をそのまま渡す」のではなく、以下の手順を踏みます。

1. **事前合意** — 1 件 3,000 円を超える支出は、使う前にメンバー間で合意しておきます。
2. **証拠の提出** — 支払いの領収書・スクリーンショットなど、金額のわかるものを提出します。
3. **確認** — 口座管理者が、使途がホワイトリストに沿っているか（＋しきい値を超える場合は事前合意済みか）を確認します。
4. **精算** — 問題なければ、口座管理者が本人へ送金します。
5. **記録** — 日付・分類・金額をこのファイルの台帳に記録します（個人名・口座情報は公開しません）。

> 本人の申告だけでは送金しません。領収書と（必要なら）事前合意がそろって初めて精算します。これは疑いをかけられないための、お互いを守る手順です。
