# End Knot

[日本語](README.md)

## About this Mod

**End Knot** is an unofficial personal fork of [Endless Host Roles (EHR)](https://github.com/Gurge44/EndlessHostRoles) for Among Us.

This mod is unofficial and is not affiliated with or endorsed by Innersloth.  
Please do not contact Innersloth regarding any issues with this mod.

Only the lobby host needs to install the mod — other players can join and enjoy the additional roles without installing anything.

Supported Among Us version: **2026.3.31**

## Key Differences from Upstream (EHR)

- **Rebrand**: Plugin GUID and mod name changed to `EndKnot` / "End Knot"
- **External communication disabled**: Update checks, achievement API, news fetching, and all other upstream network calls are disabled
- **TOHK role migration**: Roles from [TownOfHost-K](https://github.com/KYMario/TownOfHost-K) are being ported into EHR's RoleBase system
- **Calamity-themed main menu** (WIP)
- **BGM system**: In-game background music replacement (WIP)

## Installation

1. Install [BepInEx IL2CPP](https://github.com/BepInEx/BepInEx) into your Among Us folder
2. Download the latest `EndKnot.dll` from [Releases](../../releases)
3. Place `EndKnot.dll` in `Among Us/BepInEx/plugins/`
4. Launch Among Us
